# Neural Network Analysis

## Project Overview
This project demonstrates the implementation of a feed-forward neural network using TensorFlow/Keras for customer churn prediction.

## Tasks Completed
- Dataset understanding
- Data preprocessing
- Neural network model building
- Training and evaluation
- Hyperparameter experimentation
- Final reflection

## Technologies Used
- Python
- TensorFlow/Keras
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
